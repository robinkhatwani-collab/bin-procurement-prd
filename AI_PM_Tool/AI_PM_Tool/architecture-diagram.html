Project Creation Flow

Excel upload → Python server → openpyxl parsing → Claude AI generates PRD, Status Tracker, and Summary HTML → saved to disk → project hub updated automatically.

Bridge — Auto-Indexing

Every generated HTML page is automatically ingested at server startup. BS4 strips boilerplate, splits content by headings, and chunk embeddings are stored persistently in ChromaDB.

💬

RAG Chatbot Flow

User question → embedded → top-5 chunks retrieved from ChromaDB → Claude Haiku generates an answer strictly scoped to project context → answer + source chips displayed in the chat widget.
